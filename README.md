# gemma4

A VS Code extension that brings a Gemma-powered chat assistant directly into your editor — similar to Claude Code.

## Features

- **Gemma Chat Panel**: Opens a chat view in the activity bar powered by Gemma 4 via Ollama.
- Run the command **Gemma: Open Chat** from the command palette to start chatting.

## Requirements

- [Ollama](https://ollama.com) installed and running locally with the `gemma4` model pulled:
  ```
  ollama pull gemma4
  ```

## Extension Settings

This extension does not contribute any VS Code settings at this time.
